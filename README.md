# Backround-Unzipper
Unzips .zip files from a folder named Downloads to Unzipped in Documents (If you don't have these it will create them in the Home directory.

#⚠️ PYUtilities IS REQUIRED

To edit the code you will need to install my Util library of premade commands. To do this follow the instructions for installing it on the Util's page.
https://github.com/RavenSoren/PYUtilities-PYU

# How to install

The .exe is a direct program and there is no installer. To add it to your startup apps to always have it run on startup press Windows + R and type shell:startup and click enter. Then drag the .exe into the folder and restart your computer.
